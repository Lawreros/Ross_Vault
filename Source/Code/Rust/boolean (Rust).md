###### Content:
A Boolean type has two possible values: `true` and `false`. Booleans are one byte in size. The Boolean type in Rust is specified using `bool`. For example:

```rust
fn main() {
	let t = true;
	
	let f: bool = false; // with explicit type annotation
}
```
