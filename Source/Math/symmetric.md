###### Content:
 $A \in M_{n}(\textbf{F})$ is said to be symmetric if $A^T = A$

###### Properties:
- Suppose $A \in M_n(\mathbb{R})$. The matrix $A$ is symmetric if and only if $A$ is [[diagonalizable]] using a [[real orthogonal]] matrix.
