###### Content:
A function $f : \mathbb{R}^n \rightarrow \mathbb{R}$ is convex if given $x,y \in \mathbb{R}^n$ and $\alpha \in [0,1]$ $$f(\alpha x + (1-\alpha)y) \leq \alpha f(x) +(1-\alpha)f(y)$$
